# 组件

**组件**

```html
<body>
  <div id="app">
    <important-common></important-common>
  </div>
  <script type="text/javascript" src="../js/1.js"></script>
</body>
```

```js 
// 全局
Vue.component('important-common', {
	template: '<h2 style="color:red">我是Template模板, id是important-common</h2>'
})

或者局部：

var vm = new Vue({
	el: '#app',
	components: {
		'important-common': {
			template: '<h2 style="color:red">我是Template模板, id是important-common 2</h2>'
		}
	}
});

```

上面是两种组件模板template的写法，还可以把模板写在body中，然后在new Vue({})实例对象中引用：

```html
<body>
    <template id="important-common">
        <h2 style="color:red">我是Template模板, id是important-common</h2>
    </template>

    <div id="app">
      <important-common></important-common>
    </div>
    <script type="text/javascript" src="../js/1.js"></script>
</body>
```

```js
var vm = new Vue({
	el: '#app',
	template: '#important-common' # 给vue注册，不可省略
	// template: '#important-common-2'
	// ...
	// components: {
	// 	'important-common': {
	// 		template: '<h2 style="color:red">我是Template模板, id是important-common 2</h2>'
	// 	}
	// }
});
```

还有一种不常用的写法：

  </div>

```

var vm = new Vue({

```

**组件之属性Prop**

```html
<body>
<div id="app">
	<child message="hello!"></child>
</div>

<script>
// 注册
Vue.component('child', {
  // 声明 props
  props: ['message'],
  // 同样也可以在 vm 实例中像 “this.message” 这样使用
  template: '<span>{{ message }}</span>'
})
// 创建根实例
new Vue({
  el: '#app'
})
</script>
</body>
```

**组件Prop之动态属性**

```html
<body>
<div id="app">
	<div>
	  <input v-model="parentMsg">
	  <br>
	  <child v-bind:message="parentMsg"></child>
	</div>
</div>

<script>
// 注册
Vue.component('child', {
  // 声明 props
  props: ['message'],
  // 同样也可以在 vm 实例中像 “this.message” 这样使用
  template: '<span>{{ message }}</span>'
})
// 创建根实例
new Vue({
  el: '#app',
  data: {
	parentMsg: '父组件内容'
  }
})
</script>
</body>
```

**组件Prop之v-for**

```html
<body>
<div id="app">
	<ol>
    <todo-item v-for="item in sites" v-bind:todo="item"></todo-item>
  	</ol>
</div>
</body>
```

```js
Vue.component('todo-item', {
  props: ['todo'],
  template: '<li>{{ todo.text }}</li>'
})
new Vue({
  el: '#app',
  data: {
    sites: [
      { text: 'Runoob' },
      { text: 'Google' },
      { text: 'Taobao' }
    ]
  }
})
```
